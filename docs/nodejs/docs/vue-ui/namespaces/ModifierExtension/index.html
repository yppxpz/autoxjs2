weight()​

function weight(i): ModifierExt

仅在row和column直接子组件上使用有效,表示此组件占用父组件剩余空间的权重, 假如有一个row剩余高度为300，有一个子组件这设置了此修饰符为1,那么它的高度为300, 如果有两个组件都设为1，那么这两个组件各得150高度
